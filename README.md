# Ad Archive

A beautiful web application for browsing and discovering creative advertisements. Features a modern, responsive design with masonry grid layout, lightbox preview, and interactive filtering.

## Features

- **Masonry Grid Layout**: Responsive column-based layout that adapts to different screen sizes
- **Lightbox Viewer**: Click on any ad to view it in a detailed lightbox modal
- **Interactive Likes**: Like/unlike ads with persistent state
- **Sorting Options**: Sort ads by newest, oldest, popularity, or size
- **Search & Filter**: Search functionality with category filters
- **Infinite Scroll**: Lazy loading of ads as you scroll down
- **Mobile Responsive**: Optimized for desktop, tablet, and mobile devices

## Technologies

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- Google Fonts (Inter)

## Usage

Simply open `index.html` in a web browser. No build process or dependencies required.

## Design Features

- Modern gradient accents (pink to violet)
- Glassmorphism effects on overlays
- Smooth transitions and hover effects
- Clean, minimalist UI with excellent UX
- Accessible and keyboard-friendly
